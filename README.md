# Complex: Library for Complex implementation in C++

## Definitions
 - complex(a, b): a+bi
 - complex(a): a (=a+0i)
 - complex(): 0 (=0+0i)

## Functions
 - real(): z1.real()
 - imag(): z1.imag()

## Operations
 - Addition: z1+z2
 - Subtraction: z1-z2
 - Multiplication: z1*z2
 - Subtraction: z1/z2
 - IsEqual: z1==z2
 - IsNotEqual: z1!=z2
